# ZJU Cat Merge Share Card Guidelines

## Purpose

The share card is the primary viral artifact of V1. It must read clearly as a game achievement card even when viewed as a compressed image inside WeChat chat.

## Share Card Content Priority

### Must Include

- Player nickname
- Final score
- Highest cat level
- Best combo
- A clear crop of the cat stack or result-stage composition
- ZJU Cat Merge branding or title mark

### Nice to Include

- Total / weekly rank if upload succeeded quickly enough
- Short flavor headline
- Small `ФЩАґТ»°С`-style CTA is optional, not required

### Do Not Include

- Debug numbers
- Raw system errors
- Secondary controls
- Tool inventory clutter
- Dense rules text

## Card Layout Structure

### Top Zone

- Game title or compact brand mark: `ZJU Cat Merge` / `ХгґулЈсуГЁ`
- Result headline
- Optional warm decorative accent

### Middle Zone

- Main cat-stack visual or result-state composition
- This must remain the visual center of gravity
- Avoid placing heavy text over the densest part of the stack

### Bottom Zone

- Final score as the largest text block
- Highest cat and best combo as supporting stats
- Nickname and optional rank as tertiary metadata

## Visual Hierarchy

1. Score
2. Cat stack visual
3. Highest cat / combo
4. Nickname
5. Decorative copy

If hierarchy conflicts arise, protect score readability first.

## Layout Rules

- Portrait-first export
- Recommended base artboard: `1080x1920`
- Keep all critical text within the central safe area
- Leave breathing room around corners for chat-app cropping and compression
- Do not place text flush against the edges

## Safe Area Guidance

- Top safe inset: at least `120px`
- Bottom safe inset: at least `140px`
- Left / right safe inset: at least `72px`

## Style Rules

- Background should stay warm cream or soft gradient, not flat debug gray
- Use rounded stat blocks or soft separators
- Use chocolate-brown primary text on light fields
- Highlight score and peak moments with warm gold / peach accents
- Avoid full-card visual noise; let the cat art breathe

## Screenshot Capture Rules

- Capture from a polished result state, not a live messy gameplay HUD
- Hide unnecessary controls before export
- Preserve the cat stack silhouette and the emotional peak of the run
- If upload status is pending, do not wait forever; export should still work

## Rank Handling

- Show rank only if already confirmed by the backend
- If rank is delayed or upload failed, omit rank cleanly
- Never show placeholder rank values like `--` on the share card

## Export Variants

### V1 Required

- Single portrait card export

### V1 Optional

- Alternate square crop for social posting

Do not block V1 on multiple export templates.

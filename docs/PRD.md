# ZJU Cat Merge PRD

## Product Summary

`ZJU Cat Merge` is a warm, casual, physics-based merge game for Zhejiang University students and alumni. It is designed for short sessions, meme-worthy screenshots, and low-friction web distribution.

## Target Users

- Current Zhejiang University students
- Zhejiang University alumni
- Casual players in group-chat sharing scenarios

## Core User Need

Players need a low-commitment, instantly playable stress-relief game that feels cute, funny, and satisfying enough to share with friends after a strong run.

## Problem Statement

Most casual merge games either feel generic or are too heavy to share instantly in campus social contexts. This product aims to combine campus-specific identity, cute cat expression, and short-session intensity in a format that opens directly from a link.

## Goals

- Deliver a playable 2-4 minute loop
- Create screenshot-worthy outcomes with strong combo peaks
- Keep deployment and maintenance lightweight
- Support lightweight online competition with total and weekly leaderboards

## Non-Goals

- Full social network features
- Verified campus identity
- Mini Program launch in V1
- Deep monetization systems
- Heavy anti-cheat or replay review

## Core Features

- Physics-based drop-and-merge gameplay
- Forgiving red-line game over system
- Four fixed-count tools per run
- Anonymous nickname-based leaderboard system
- Screenshot-first result sharing
- Audio feedback and prototype-stage BGM

## Experience Pillars

- `Soft and cute`: warm palette, rounded interface, expressive cat assets
- `Chaotic and funny`: unstable stacks, shake/bomb recovery, weird cat tower moments
- `Relieving but tense`: forgiving fail state with danger pressure
- `Shareable`: results feel worth showing to classmates and friends

## Success Metrics

- Players can start a run in seconds from a shared link
- A first-time player understands the loop without tutorial overload
- Strong runs generate clear screenshot-sharing intent
- Leaderboards update reliably after valid runs

## Open Constraints

- Final public BGM must use properly licensed assets
- Cat source GIFs must be converted into standardized transparent PNG runtime assets
